Free Code Camp Simon® Game

Instructions/Rules

  • Select Strict on/off, then press Start to begin.
  • Simon will show you which buttons to click; do what Simon says.
  • If Strict off, user gets two more attempts after making a mistake.
  • If Strict on, game is sudden death.
  • Strict mode cannot be changed once user presses Start.
  • Press Reset at any time to start over again.
  • Complete 20 rounds to win game.
